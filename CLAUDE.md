# CLAUDE.md

This repo (`vnre`) carries a synced copy of David Van Noy Jr.'s Claude operating
brain. **Read [`claude/CLAUDE.md`](claude/CLAUDE.md) first** — it's the root
operating manual (who DVN is, how to respond, the project/reference structure,
standing workflows).

The canonical source of these files is Google Drive (`_Claude md/`). This repo is
the GitHub-hosted mirror so Claude Code on the web can pick up the manual
automatically. See [`claude/README.md`](claude/README.md) for what's included,
what's deliberately left in Drive, and how the two stay in sync.
