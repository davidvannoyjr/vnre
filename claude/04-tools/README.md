# Tools

Code and integrations. Not content — don't save deliverables here.

## Contents
- `followupboss-mcp/` — Node MCP server for Follow Up Boss CRM. Contains `fub-key.local.md` (API key).
- `vnre-dashboard/` — Node app. See its `SETUP.md`.

## Sync rule
`node_modules` is intentionally NOT stored here (sync killer). After cloning/syncing to a new device, run `npm install` inside each app folder before first use. Never sync `node_modules` to Google Drive.

> Note: the actual tool source (followupboss-mcp, plp-presentation-builder, the skill folders, vnre-dashboard) lives in Google Drive under `04 Tools/` and is intentionally NOT mirrored to this repo — see `/claude/README.md`.
