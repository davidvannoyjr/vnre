A listing description should take you two minutes, not twenty. Here's the system.

## The prompt

Copy this. Fill the brackets. That's it.

```
You're writing an MLS listing description in my voice: confident, specific,
no hype, no clichés ("stunning," "must see," "won't last"). Use ONLY these
facts — never invent features.

Property: [beds/baths, sqft, style, neighborhood, city]
Standouts: [3–5 real features — updates, lot, location, schools]
Buyer: [who this home fits]
Length: 140–160 words.

Lead with the single best thing. End with one line of urgency that's true.
```

## Why it works

- **"In my voice" + banned words** kills the generic real-estate-bot tone.
- **"Use ONLY these facts"** stops it from hallucinating a "wine cellar" you never mentioned — a compliance and trust problem.
- **A length cap** keeps it MLS-ready instead of a three-paragraph essay.

## Tighten it in one move

If the first draft is 80% there, don't re-prompt from scratch. Say: *"Cut it to 150 words, make the opening line punchier, keep everything else."* Editing beats regenerating.

## Make it permanent

Save this as a **Saved Prompt** (or a Custom GPT — covered in the intermediate track). Now it's a 90-second task every listing, forever.

## Your win

Take your next live listing. Run it through. Compare to what you'd have written. Use the better one. You just bought back 18 minutes per listing.
