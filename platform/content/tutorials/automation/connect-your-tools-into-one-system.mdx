If you're copying data from one app into another by hand, you're doing robot work. Automation hands that to a robot so you get back to dollar-productive activity.

## The mental model

An automation is three parts: **Trigger → Action(s)**.

> *When* a new lead hits my CRM (trigger), *then* tag it, send a text, and add me a task to call within 5 minutes (actions).

Tools like **Zapier**, **Make**, and **n8n** wire these together with no code. Your CRM (**Follow Up Boss**) has built-in action plans that cover a lot of it.

## The first one to build

Forget the fancy stuff. Build a **speed-to-lead** automation:

1. **Trigger:** new lead created in Follow Up Boss.
2. **Action:** auto-text the lead a short, human first response.
3. **Action:** create a task for you to call within 5 minutes.
4. **Action:** if no contact in 48 hours, drop them into a nurture sequence.

Speed-to-lead is the single highest-ROI automation in real estate. The agent who responds first usually wins. Let the system guarantee you're first while you're on a listing appointment.

## Don't over-build

The trap is spending a weekend wiring 14 zaps you'll never trust. Build one. Watch it run for a week. Then build the next. Systems compound; tinkering doesn't.

## Your win

Map your speed-to-lead flow on paper — trigger, then each action. The intermediate track builds it live in Follow Up Boss.
